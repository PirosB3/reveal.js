What is Sentiment Analysis

  • It's software for automatically extracting opinions, emotions and sentiment in text
  • It allows us to track attitudes and feelings on the web. People write blog posts, comments, reviews and tweets about all sorts of different topics.
  • It allows us to track products, brands, and people to determine whether they are viewed positively or negatively on the web (Marie-Claire Jerkins)

Microblogging websites have evolved to become a source of varied kind of information

  • People post real time messages about their opinions on a variety of topics (politics, weather, fashion, ..)

  • Companies have started to use this information to get a sense of general sentiment for their product

WorldMood

WorldMood is a real-time sentiment analysis processor, that uses natural language processing combined with machine learning to provide aggregated statistics and insights to clients on a topic or trend.

  • Real-time feeds are processed instantly
  • Accessible on any web browser, with stunning user experience
  • Scalable and maintainable architecture, future-proof!

Platform Overview

Platform Overview

  • Streamer

    Provides a non-blocking pipeline for collecting real-time tweets from Twitter

  • Classifier

    Extracts informative features from phrases and predicts their sentiment

  • Web Application

    Receives classified tweets in real-time and displays informative features through graphs and charts

  • Core

    Provides a centralized process the above services use to communicate

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the background.

Custom Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name. 


Reveal.addEventListener( 'customevent', function() {
	console.log( '"customevent" has fired' );
} );

Slide Backgrounds

Set data-background="#007777" on a slide to change the full page background to the given color. All CSS color formats are supported.

Background Transitions

Pass reveal.js the backgroundTransition: 'slide' config argument to make backgrounds slide rather than fade.

Background Transition Override

You can override background transitions per slide by using data-background-transition="slide".
